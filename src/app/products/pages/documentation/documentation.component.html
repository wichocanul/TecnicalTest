Documentación

Tecnologías utilizadas

Este proyecto fue realizado con Angular en su versión 14.2.5. Para la creación del proyecto se utilizó Angular CLI y eventualmente VisualStudio Code como editor de código, y GitHub como gestor de versiones.

Modulos y Componentes

Se crearon dos módulos en este proyecto, dentro de la carpeta ./src/ encontraremos una con el nombre de Products y otra con el nombre de shared.

En la carpeta shared/ encontraremos todos los componentes que la mayoría de las pestañas utilizaran, estos componentes son: el menú y el footer. Cada una contará con su archivo .CSS .HTML y .TS

En la carpeta de /productos tendremos una carpeta con el nombre de pages, ahí encontraremos todos los componentes que tendrán su propia ventana, utilizando los componentes de shared, al igual que el modulo anterior estos contarán con su archivo .CSS .HTML y .TS

Rutas

Para las rutas se creo un archivo llamado app-routing.module.ts el cual utiliza los módulos de Angular import [ RouterModule, Routes ]; from "@angular/router"; en la cual solo se agregaron 3 rutas.

  • La pagina principal (Home)
  • La parte de la documentación (Doc)
  • El Error 404 page not found(**)

Esta ultima solo se podrá ver cuando el usuario edite la URL y la aplicación no encuentre esa ruta como se muestra en la siguiente imagen.

Servicios

Se implemento un archivo en la carpeta Services con el nombre producto.service. Este archivo es el encargado de consultar la API y enviarla a la página principal, esto creando una interfaz de tipo RestProductsResponse y así poder maquetar cara resultado en la tabla principal.
